# worklog

`worklog` — read-only CLI-утилита для получения краткого отчета о работе разработчика в Git-репозитории.

Утилита показывает, над какими ветками пользователь работал, какие коммиты и stash создавал, а также какие файлы были затронуты.

## Команды

```bash
worklog today
worklog yesterday
worklog status
```

### `worklog today`

Показывает активность за сегодняшний календарный день:

- собственные коммиты;
- файлы из этих коммитов;
- stash, созданные сегодня;
- файлы из этих stash;
- текущие незакоммиченные файлы.
